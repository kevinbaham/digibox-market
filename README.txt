📄 README - Pack CV DigiBox

✅ Ce dossier contient 100 modèles de CV prêts à l'emploi.
- Ouvrez n'importe quel fichier cv-XX.html dans votre navigateur.
- Cliquez sur les zones pour éditer directement (nom, compétences, expériences).
- Cliquez sur « Imprimer / Enregistrer en PDF » pour générer votre CV final.

📁 Structure :
- assets/logo.png : votre logo (vous pouvez le remplacer)
- assets/icons/ : icônes contact
- styles/cv-XX.css : styles spécifiques

✅ Recommandé : utiliser Chrome ou Edge pour le rendu parfait.
